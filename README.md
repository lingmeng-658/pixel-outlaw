# Pixel Outlaw

一个网页可玩的西部牛仔像素风俯视角射击小游戏。

本项目受《草原大王的冒险》的玩法启发，但会使用自己的名字、素材和设定。

## 游戏目标

玩家在荒野场景中移动、射击、躲避敌人、击败敌人并获得分数，尽可能存活更久。

## v0.1 目标：30 秒试玩版

- 标题界面
- Start 按钮
- WASD 控制玩家移动
- 方向键四向射击
- 敌人从屏幕边缘生成并靠近玩家
- 子弹击中敌人后敌人消失
- 玩家被敌人碰到会扣血
- 显示生命值和分数
- 死亡后显示 Game Over
- 支持重新开始
