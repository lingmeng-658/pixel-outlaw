# 一、本轮目标

把当前作为区域切换雏形的 Town Road 实现为第二关第一版。

第二关的核心主题是：

```txt
持续增压战斗 + 新敌人行为 + 可争夺强化道具
```

本轮需要完成一个可以从第一关正常进入、可以完整打通、可以保存和重新开始、可以由用户真实试玩调试的第二关版本。

重点不是一次锁死最终平衡，而是把已确认的行为完整接通，并将主要数值集中配置，方便后续试玩调整。

- 实体障碍物、掩体、视线遮挡或寻路系统

- 第三关内容

- 清屏道具
- 枪手的新职业分支
- 移动端触屏操作

- 把第一关 Continue 改回阶段级恢复
- 自动修改无关依赖

第二关保持开放战场。可以增加不参与碰撞的道路、栅栏、马车、石堆等装饰，但不要引入会阻挡玩家、敌人或子弹的实体障碍。

# 四、第二关连续遭遇战

## 1. 玩家看不到明确波次

第二关不显示 WAVE 1、WAVE CLEAR 等明显波次 UI。

底层仍使用隐藏批次组织敌人，但玩家感受到的是持续补怪和不断增压。

## 2. 下一批接续条件

只有当前隐藏批次的敌人已经全部生成后，才开始判断下一批。

满足任一条件即可激活下一批：

```txt
A. 当前批次最后一个敌人生成后经过 2.5 秒
B. 当前批次已经消灭 65%
```

这里的消灭比例按当前批次全部成员计算，包括该批次中的特殊敌人。

以上为第一版测试值，必须配置化。

## 3. 动态场上上限

下一批激活不代表一次性生成全部敌人。

达到当前阶段的存活敌人上限后：

- 暂停继续生成
- 敌人死亡后继续从待生成队列补入
- 不丢失或跳过任何计划敌人

第一版测试上限：

```txt
批次 1：10
批次 2：12
批次 3：14
批次 4：16
批次 5：18
```

## 4. 第一版敌人总量

第一关固定流程约 29 个普通敌人，第二关目标是约 2 到 2.5 倍的小怪，再加入有限特殊敌人。

第一版使用以下可调配置：

| 隐藏批次 | 普通追踪者 | 冲锋者 | 枪手 | 存活上限 |
|---|---:|---:|---:|---:|
| 1 | 12 | 1 | 0 | 10 |
| 2 | 14 | 1 | 1 | 12 |
| 3 | 14 | 2 | 1 | 14 |
| 4 | 12 | 1 | 0 | 16 |
| 5 | 16 | 1 | 2 | 18 |

合计：

- 普通追踪者 68
- 直接生成冲锋者 6
- 枪手 4
- 最终阶段同时出现 2 个枪手

这是试玩基线，不是最终平衡。

## 5. 最终阶段

最终阶段两个枪手应从不同方向进入。

它们的攻击窗口需要错开约 0.4 到 0.7 秒，避免完全同步形成无法阅读的弹墙。

同一时刻最多允许一个枪手处于强化攻击的“第二次自适应补射阶段”。目标是交替形成持续压力，而不是同时倾泻不可躲避弹幕。

当全部计划敌人已经生成且场上敌人清空时，第二关完成。

# 六、冲锋者

## 1. 基础行为

冲锋者循环：

1. 正常靠近玩家
2. 进入准备距离
3. 显示清晰蓄力和冲锋方向预警
4. 锁定方向
5. 沿直线高速冲锋
6. 冲锋结束后短暂恢复
7. 再次进入追踪

第二关中冲锋方向一旦锁定就不能中途转向。

玩家应能通过观察预警后侧移来躲避。

## 2. Coffee 强化

- 普通追踪者捡到 Coffee：转化为冲锋者
- 冲锋者捡到 Coffee：获得更快的移动、冲锋或更短恢复时间
- 强化效果必须有明显视觉反馈
- 数值配置化，不要做到无法反应

## 3. Dynamite 强化

冲锋者捡到 Dynamite 后成为爆破冲锋者：

- 保留直线冲锋
- 撞到玩家、到达冲锋终点、撞到世界边界或死亡后触发爆炸
- 爆炸前显示清晰范围与短预警
- 爆炸可以伤害附近敌人，允许玩家诱导它清理敌群
- 不增加中途转向；转向能力留到第三关

# 七、枪手

## 1. 基础移动

枪手不直接贴脸追踪。

第一版需要：

- 与玩家保持一个可配置的作战距离
- 太近时后退
- 太远时靠近
- 合适距离时移动、瞄准和射击
- 避免停在地图极端死角

不需要障碍物寻路。

## 2. 普通二连发

普通枪手每轮攻击发射两颗子弹：

1. 第一发瞄准玩家当前所在位置
2. 第二发瞄准一个基于玩家当前速度的短距离预测位置

两个目标应在瞄准阶段锁定。发射后不能持续追踪玩家，也不能每帧重新计算。

攻击前必须有可读的瞄准或枪口提示。

## 3. Ammo 强化：自适应四连发

枪手捡到 Ammo 后，攻击变为两组二连发：

### 第一组

- 当前所在位置弹
- 初始移动预测弹

### 观察窗口

- 第一组结束后观察玩家约 0.22 秒
- 读取这段时间内玩家的实际位移，不只读取最后一帧按键
- 观察结束时同时锁定第二组两个目标
- 锁定后给约 0.10 秒重新瞄准提示
- 锁定后不再追踪

### 第二组

- 沿玩家第一次闪避方向的短预测弹
- 同一方向的更远预测弹

玩家的反制方式应是：第一组后移动一次，枪手重新瞄准后再做第二次变向。

## 4. Buckshot 强化：两轮扇形三发

枪手捡到 Buckshot 后，每轮攻击分两次发射：

### 第一轮

- 发射一个较宽的扇形三发
- 三颗子弹之间留下可辨认缝隙
- 作用是迫使玩家选择穿过某个空隙或绕开弹幕

### 观察窗口

- 观察玩家约 0.22 秒
- 判断玩家实际选择的移动方向和第一轮弹幕空隙
- 锁定第二轮方向
- 给约 0.10 秒枪口调整提示
- 锁定后不再追踪

### 第二轮

- 再发射扇形三发
- 扇形可以略窄于第一轮
- 大概率覆盖玩家刚才选择的空隙和继续移动路线

玩家仍应能通过第二次变向或折返躲开。

## 5. 强化限制

- 枪手的 Ammo 与 Buckshot 强化互斥
- 第一版同时最多一个强化枪手
- 两个最终枪手不能同步执行第二次自适应补射
- 强化状态必须有明显外观、枪口或颜色反馈

# 八、第二关道具

## 1. 道具分类

### 玩家专属生存道具

- Heart / Potion
- Shield

敌人不能：

- 主动争夺
- 顺路拾取
- 通过碰撞销毁

### 可争夺战斗强化

- Coffee
- Ammo Box
- Buckshot Crate
- Dynamite

## 2. 玩家效果

### Coffee

沿用现有效果：短时间提高移动速度。

### Ammo Box

- 短时间明显提高射速
- 不修改第一关基础射速
- 效果数值配置化

### Buckshot Crate

- 短时间把玩家每次射击变为扇形三发
- 每次按键仍视为一次射击动作
- 与 Ammo 作为武器模式互斥，拾取新的模式时替换旧模式并刷新持续时间

### Dynamite

Dynamite 是命中附魔，不是只允许发射有限颗子弹。

- 获得若干层爆破装药
- 子弹有效命中敌人时消耗一层并产生小范围爆炸
- 射空不消耗
- 爆炸伤害附近敌人，不伤害玩家
- 可以与 Coffee、Ammo 或 Buckshot 共存
- Buckshot 一次按键发出的三颗子弹最多消耗一层 Dynamite，避免一轮瞬间消耗三层

第一版层数和范围配置化。

## 3. 敌人效果

| 道具 | 合格敌人 | 敌人效果 |
|---|---|---|
| Coffee | 普通追踪者、冲锋者 | 追踪者转化为冲锋者；冲锋者进一步加速 |
| Dynamite | 普通追踪者、冲锋者 | 转化为爆破追猎者或爆破冲锋者 |
| Ammo | 枪手 | 两组自适应二连发 |
| Buckshot | 枪手 | 两轮自适应扇形三发 |

枪手第一版忽略 Coffee 和 Dynamite；近战敌人忽略 Ammo 和 Buckshot。

# 九、爆破追猎者

普通追踪者捡到 Dynamite 后：

1. 继续追踪玩家
2. 接近到配置距离后停下
3. 点燃引线
4. 地面显示爆炸范围
5. 短暂倒计时后爆炸

如果被玩家提前击杀：

- 不要无预警瞬间爆炸
- 进入约 0.4 到 0.6 秒的快速引爆
- 显示范围
- 爆炸伤害附近敌人

这个敌人既是威胁，也是玩家可以利用的战术资源。

# 十、道具争夺系统

## 1. 保证一个主要抢夺者

每次可争夺道具落地后：

- 从当前存活、能使用该道具、未处于不可打断动作的合格敌人中均匀随机选择一个主要抢夺者
- 不固定选择最近或最危险敌人
- 选择范围只包含当前可用且可到达的候选者
- 主要抢夺者获得清晰 `!`、闪光、描边或短暂连线提示

如果计划刷新道具时没有合格敌人：

- 延迟刷新到合格敌人出现
- 不生成一个无人能争夺的“假教学”道具

## 2. 重新分配

主要抢夺者死亡或失效，而道具仍存在时：

- 从剩余合格敌人重新随机分配
- 玩家拿走道具后立即清理所有抢夺状态

## 3. 顺手拾取

没有被指定为主要抢夺者的合格敌人，如果：

- 与道具距离足够近
- 当前没有执行锁定攻击、冲锋或其他不可打断动作

也可以临时绕行并拾取。

最终拾取动作可有很短的预留或占用状态，避免大量敌人同时挤在同一位置；但玩家仍然可以抢走。

## 4. 玩家优先

玩家和敌人同一帧接触道具时，优先保证玩家拾取，避免物理回调顺序造成不公平。

## 5. 枪手争夺行为

枪手争夺 Ammo 或 Buckshot 时：

- 玩家较远：靠近道具并尝试拾取
- 玩家开始逼近：尽量站在玩家与道具之间，使用普通攻击阻止直线靠近
- 玩家非常近：优先自保或压制，再尝试拾取
- 最终拾取动作约 0.4 到 0.6 秒，期间不能射击，可以被玩家击杀或抢走道具打断

## 6. 刷新位置与落地提示

可争夺道具：

- 不刷在地图角落
- 不刷在玩家或敌人脚下
- 不刷在出生边缘
- 保证双方都看得见且能接近
- 落地前给约 0.5 到 0.8 秒提示
- 落地后约 0.35 到 0.6 秒再让主要抢夺者正式行动，给玩家一次反应机会

同一时间最多存在一个尚未拾取的可争夺强化道具。

# 十一、强化道具出现规划

强化道具不能完全依赖随机掉落，否则可能无法展示核心机制。

使用：

```txt
规划出现时机 + 合法区域内随机位置 + 随机主要抢夺者
```

第一版总量：

| 道具 | 数量 |
|---|---:|
| Coffee | 3 |
| Dynamite | 3 |
| Ammo | 2 |
| Buckshot | 2 |
| 合计 | 10 |

推荐顺序：

1. 前段 Coffee：展示近战敌人争夺和追踪者转化为冲锋者
2. 前段 Dynamite：展示爆破追猎者和敌人爆炸伤害
3. 第一名枪手出现后 Ammo：展示普通二连发与强化四连发差异
4. 下一名枪手出现后 Buckshot：展示两轮扇形补空
5. 后半段开始混合重复出现 Coffee / Dynamite / Ammo / Buckshot
6. 最终双枪手阶段错开出现一次枪手道具与一次近战道具

不要让 Coffee 与 Dynamite、Ammo 与 Buckshot 同时落地形成两个未拾取争夺事件。

早期四次是经过设计的展示；后续出现时间和合法落点可以在进度窗口内随机。

# 十二、回复道具、Shield 与金币

## 1. Heart / Potion 概率掉落

第二关不再使用第一关式固定回血教学刷新。

敌人死亡时只有满足以下条件才参与回复掉落判定：

- 玩家未满血
- 场上没有未拾取 Heart
- 回复掉落系统不在硬性关闭状态

掉落后进入防连续掉落冷却。

第一版建议配置：

- 正常冷却约 8 到 12 秒
- 同时要求至少再击杀约 4 到 6 个合格敌人
- 生命越低，基础掉率越高

时间和击杀数两个条件都满足后才重新开放正常掉落。

## 2. 低血“及时雨”结算

当同时满足：

- 玩家只剩 1 HP
- 场上没有未拾取 Heart
- 当前回复掉落仍在冷却
- 本次 1 HP 状态尚未触发过及时雨

则：

- 将当前剩余冷却缩短到约 1.5 到 2 秒
- 将最低击杀要求降低到 1
- 提高接下来三次合格击杀的掉率
- 第三次仍未掉落则保底生成 Heart

玩家恢复到 2 HP 以上后，再次掉到 1 HP 时才允许重新触发。

Heart 始终玩家专属。

## 3. Shield

第二关第一版在最终高压阶段前规划生成一个玩家专属 Shield，提供稳定容错。

敌人不能争夺、拾取或销毁 Shield。

## 4. 金币

金币掉落是独立通道：

- 不受 Heart 冷却影响
- 不受强化道具是否在场影响
- 可以与其他掉落同时发生
- 第二关具体概率或目标数量配置化

不要实现商店。

第二关全部敌人生成完毕且场上清空后：

# 十七、验收标准

## 构建

```bash
npm run build
```

必须通过。若失败，先解释原因，再做最小修复。

## 第一关回归

1. 标题 Start / Continue 正常
2. 第一关敌人和道具教学不变
3. 第一关可正常通关并进入 Town Road
4. Dusty Outskirts 与 Town Road 往返不出错
5. 第一关关卡级 Continue 不回归

## 连续遭遇战

1. 不显示明确波次 UI
2. 当前批次生成完后约 2.5 秒会接入下一批
3. 快速消灭约 65% 时会提前接入
4. 达到动态上限后停止补怪
5. 杀敌后继续从队列补入
6. 不会一次性生成全部敌人
7. 最终两个枪手同时存在且攻击错峰
8. 全部计划敌人清空后完成关卡

## 敌人

1. 冲锋者有清晰预警、锁定、直线冲锋和恢复
2. 第二关冲锋者不会中途转向
3. 枪手保持距离并使用普通二连发
4. Ammo 枪手第二组读取第一次闪避，但锁定后不追踪
5. Buckshot 枪手第一轮留缝，第二轮补玩家选择的空隙
6. 两枪手不会同步形成不可阅读弹墙
7. 爆破敌人爆炸可以伤害其他敌人

## 道具争夺

1. Heart 和 Shield 永远不能被敌人拾取或销毁
2. 每个可争夺道具有一个随机合格主要抢夺者
3. 主要抢夺者死亡后可以重新分配
4. 非主要抢夺者靠近时可以顺手拾取
5. 玩家与敌人同时接触时玩家优先
6. 没有合格敌人时延迟刷新对应道具
7. 同时最多一个未拾取争夺道具

## 玩家强化

1. Ammo 明显提高射速
2. Buckshot 产生扇形三发
3. Ammo 与 Buckshot 互相替换
4. Dynamite 只在有效命中时消耗
5. 射空不消耗 Dynamite
6. Buckshot 一次按键最多消耗一层 Dynamite

## 回复与奖励

1. Heart 按概率掉落且不会连续爆出
2. 1 HP 且场上无 Heart 时及时雨会缩短当前冷却
3. 第三次合格击杀仍未掉落时触发保底
4. 金币不受回复冷却限制
5. 最终阶段前有玩家专属 Shield
6. 第二关首次通关最大生命 +1，当前生命 +1
7. 重打不重复奖励

## 存档与重入

1. 第二关中途 Continue 从 Town Road 开头重来
2. 不恢复中途战场和临时 Buff
3. 第二关完成状态和最大生命奖励能保存
4. 旧存档可迁移
5. New Game 不继承旧进度
6. 离开 Town Road 后不再后台刷怪
7. 重进不会叠加计时器和碰撞
